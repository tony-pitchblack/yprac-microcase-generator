Ты получаешь список ошибок студента в JSON.
Для каждой ошибки сгенерируй микро-кейс. Ты должен сгенерировать следующие пункты: 
- Условие задачи (Короткое описание данных)
- Ввод в программу (То что нужно подать на вход программе)
- Вывод из пограммы (То что должна вывести программа, после работы)
Офрмление должжно быть в JSON:
[
  {{
    "task_condition": "описание задачи",
    "output": "Данные, которые нужно положить в программу",
    "input": "Что должно быть на выходе из программы",
  }}
]

Ошибки:
{errors}
